## 1. В чём проблема

Веб-чат с ИИ не помнит тебя между сессиями.
Закрыл вкладку → открыл новую → ИИ «проснулся чистым».
Это не баг, а осознанное решение ради приватности и экономии.

---

## 2. 4 способа сохранить память

| Способ | Сложность | Где работает | Что помнит |
|--------|-----------|-------------|------------|
| Текстовый файл | ★☆☆ | Веб-чат | Всё (если прикрепить) |
| Локальный помощник с API | ★★☆ | Терминал | Всё навсегда |
| Системный промпт | ★★☆ | Локальный скрипт | Ключевые факты |
| Импорт сессии | ★★☆ | Локальный помощник | Конкретный диалог |

---

## 3. Способ 1: Текстовый файл

Создай файл с описанием проекта. Прикрепляй его в новом чате.
Плюс: просто. Минус: надо прикреплять вручную.

---

## 4. Способ 2: Локальный помощник

Скрипт `assistant.py` хранит всю историю в `chat_log.json`.
При каждом запуске загружает прошлые диалоги и помнит всё.
Нужен Python и API-ключ OpenRouter.

---

## 5. Способ 3: Системный промпт

Вшивает знания о проекте прямо в запросы к ИИ.
Не стирается, но не помнит сами диалоги.

---

## 6. Способ 4: Импорт веб-сессии

После важного разговора говоришь:
«Подготовь import_session.py с нашей сессией»
ИИ даёт скрипт, ты запускаешь — всё добавляется в память.

## 8. Частые ошибки

| Ошибка | Решение |
|--------|---------|
| 401 | Проверить ключ в .env |
| 402 | Пополнить счёт или перейти на OpenRouter |
| 404 | Использовать openrouter/free |
| API_KEY пустой | Проверить python-dotenv |
| Ответ None | Повторить запрос |

## 10. Лучшая стратегия

1. Настрой `assistant.py` — вечная память в терминале
2. Держи `project_context.txt` для веб-чата
3. После важных диалогов: «Подготовь import_session.py»
4. Обновляй системный промпт при изменениях в проекте

---

## 11. Веб-промпт и карточный стиль (v3.0)

Файл `web_prompt.txt` задаёт правила поведения ИИ в веб-чате.
**Важное правило:** всегда выдавай **полный код файла**, а не фрагменты.

### Сохранение при выходе
Когда пользователь прощается, ИИ предлагает сохранить сессию (Да/Нет).
При согласии выдаётся **полный комплект из 5 пунктов**:
1. `import_session.py` (весь диалог от начала)
2. `web_prompt.txt`
3. `project_context.txt`
4. `Гайд_Память_для_ИИ.md`
5. Напоминание про `chat_log.json`

### Завтра
Прикрепи `web_prompt.txt` + `project_context.txt` + `chat_log.json` + `Гайд_Память_для_ИИ.md`.
