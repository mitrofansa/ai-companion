# QASAR Hub

3D-платформа ИИ-компаньона нового поколения.

## Что это
QASAR — это персональная среда обитания для искусственного интеллекта. Пользователь взаимодействует с ИИ не через текстовое окно, а через полноценное 3D-пространство с голографическими интерфейсами, аватаром и голосовым управлением.

## Текущая версия
**release-0.1** (эталон, 12 мая 2026)

## Возможности v0.1
- 3D-комната (40м, 12 окон, модель из Blender)
- Два цилиндра-стекла
- Панорама города 360°
- 3 LED-кольца + плафон
- Голографический HUD с интерактивными часами
- Управление: орбита + WASD

## Стек
Three.js + Vite + Blender + Electron (планируется)
